# LedgerMind

LedgerMind is a **read-only**, **local-first** assistant that connects to [YNAB](https://www.ynab.com/) and helps you reason about debt payoff, savings goals, cash flow, and budget tradeoffs. It is **not** tax, legal, or investment advice.

- **CLI**: `ledgermind doctor`, `list-budgets`, `snapshot`, and more ([setup](setup.md)).
- **MCP**: read-only tools via `ledgermind run-mcp` ([mcp-tools](mcp-tools.md)); **skill** for agents: [skills](skills.md).
- **Source of truth**: Implementation brief in [planning/initial_plan.md](planning/initial_plan.md).
- **Release / future**: [Phase 5 (release readiness)](planning/phase5_release.md), [Phase 6 (future design)](planning/phase6_future.md).
